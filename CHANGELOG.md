# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-29

Initial public release.

### Added

- `jsonspecs init <project-name>` for scaffolding a new rules project
- `jsonspecs studio` for local studio mode
- `jsonspecs validate` for static validation of a rules project
- `jsonspecs build` for snapshot assembly
- project-local Node operator pack loading from `manifest.json`
- support for the `jsonspecs` operator contract based on `ctx.get()` / `ctx.has()`
- initial automated CI workflow
